Quick Start

The shortest path to value: connect a sender, scrape leads, build a campaign, send outreach, and track replies.

1

Connect email

Add your first sender account with SMTP/IMAP, then verify it's ready to send.

2

Scrape leads

Use the lead scraper to pull local businesses by city + niche and save them to a list.

3

Build campaign

Create a campaign, add subject/body, optional variants, follow-ups, and schedule.

4

Launch + track

Send, monitor replies in the unified inbox, and move hot leads into the pipeline.

Integration Guides

These are the core integrations that make LeadRipper feel like an outbound operating system rather than a simple email tool.

Twilio

Connect calling infrastructure

  • Create or connect a Twilio account
  • Attach a number
  • Configure softphone / call routing
  • Test outbound call flow
GoHighLevel

Sync contacts and opportunities

  • Connect API key or location credentials
  • Map lead and opportunity fields
  • Choose when sync happens
  • Prevent duplicates
ElevenLabs

Enable AI voice

  • Add API key
  • Pick a voice or agent
  • Link Twilio call flow
  • Test voice quality + fallback handling

Campaign Setup Checklist

Campaigns should be teachable. The checklist below can also be turned into in-app onboarding and tooltips.

Recipients

Build the right audience

Filter by list, city, niche, verified emails only, and exclude recently emailed leads.

Message

Write a strong offer

Use merge tags, save templates, add A/B variants, and make the first email clear and relevant.

Sequence

Follow up properly

Add follow-up steps, delays, and send schedules so campaigns feel deliberate instead of spammy.

Tip: keep the lead scraper as the hero, and treat campaigns, inbox, pipeline, calling, and integrations as clean supporting modules.
