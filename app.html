    API Documentation
    Endpoint: POST https://leadripper.com/.netlify/functions/cli-api
    Auth: X-Api-Key: YOUR_KEY
    Body: { "action": "action_name", ...params }

    Actions: stats, get_leads, search, update_lead, delete_lead, create_list, add_to_list, bulk_no_email_list, call_lead, call_list, get_call_logs, score_website, get_phone_numbers, get_agents, assign_phone

    Example:
    curl -X POST https://leadripper.com/.netlify/functions/cli-api \
  -H "Content-Type: application/json" \
  -H "X-Api-Key: YOUR_KEY" \
  -d '{"action":"get_leads","city":"Miami","industry":"plumbing","limit":10}'
